# 🔐 Configuration CivicFix\n\n## Variables d'environnement\n\nCe guide documente toutes les variables d'environnement utilisées par CivicFix.\n\n## Django Settings\n\n### SECRET_KEY\n**Type:** String  \n**Description:** Clé secrète pour signer les sessions et tokens  \n**Défaut:** (requis)\n\n```env\nSECRET_KEY=django-insecure-your-secret-key-change-in-production\n```\n\n**Générer une nouvelle clé:**\n```bash\npython -c 'from django.core.management.utils import get_random_secret_key; print(get_random_secret_key())'\n```\n\n### DEBUG\n**Type:** Boolean  \n**Description:** Mode debug (affiche les erreurs détaillées)  \n**Défaut:** False  \n**Valeurs:** True, False\n\n```env\n# Développement\nDEBUG=True\n\n# Production\nDEBUG=False\n```\n\n⚠️ **ATTENTION:** Toujours FALSE en production!\n\n### ALLOWED_HOSTS\n**Type:** String (comma-separated)  \n**Description:** Domaines autorisés à accéder l'application  \n**Défaut:** localhost,127.0.0.1\n\n```env\n# Développement\nALLOWED_HOSTS=localhost,127.0.0.1\n\n# Production\nALLOWED_HOSTS=yourdomain.com,www.yourdomain.com,api.yourdomain.com\n```\n\n## Base de données\n\n### DB_ENGINE\n**Type:** String  \n**Description:** Backend de base de données  \n**Défaut:** sqlite3  \n**Valeurs:**\n- `django.db.backends.sqlite3` - SQLite (dev)\n- `django.db.backends.postgresql` - PostgreSQL (prod)\n- `django.db.backends.mysql` - MySQL\n\n```env\n# SQLite (Développement)\nDB_ENGINE=django.db.backends.sqlite3\nDB_NAME=db.sqlite3\n\n# PostgreSQL (Production)\nDB_ENGINE=django.db.backends.postgresql\nDB_NAME=civicfix_db\nDB_USER=civicfix_user\nDB_PASSWORD=your_password\nDB_HOST=localhost\nDB_PORT=5432\n```\n\n### DB_NAME\n**Type:** String  \n**Description:** Nom de la base de données (ou chemin pour SQLite)\n\n### DB_USER\n**Type:** String  \n**Description:** Utilisateur PostgreSQL/MySQL\n\n### DB_PASSWORD\n**Type:** String  \n**Description:** Mot de passe de la base de données\n\n### DB_HOST\n**Type:** String  \n**Description:** Adresse du serveur de BD  \n**Défaut:** localhost\n\n### DB_PORT\n**Type:** Integer  \n**Description:** Port de la BD  \n**PostgreSQL défaut:** 5432  \n**MySQL défaut:** 3306\n\n## Email\n\n### EMAIL_BACKEND\n**Type:** String  \n**Description:** Service d'envoi d'emails  \n**Défaut:** console (affiche dans le terminal)\n\n```env\n# Développement (affiche dans console)\nEMAIL_BACKEND=django.core.mail.backends.console.EmailBackend\n\n# Production (SMTP)\nEMAIL_BACKEND=django.core.mail.backends.smtp.EmailBackend\nEMAIL_HOST=smtp.gmail.com\nEMAIL_PORT=587\nEMAIL_USE_TLS=True\nEMAIL_HOST_USER=your-email@gmail.com\nEMAIL_HOST_PASSWORD=your-app-password\n```\n\n### EMAIL_HOST\n**Type:** String  \n**Description:** Serveur SMTP\n\n**Exemples:**\n- Gmail: `smtp.gmail.com`\n- Outlook: `smtp-mail.outlook.com`\n- SendGrid: `smtp.sendgrid.net`\n\n### EMAIL_PORT\n**Type:** Integer  \n**Description:** Port SMTP  \n**Défaut:** 587 (TLS) ou 465 (SSL)\n\n### EMAIL_USE_TLS\n**Type:** Boolean  \n**Description:** Utiliser TLS pour chiffrement\n\n### EMAIL_HOST_USER\n**Type:** String  \n**Description:** Adresse email d'authentification\n\n### EMAIL_HOST_PASSWORD\n**Type:** String  \n**Description:** Mot de passe d'authentification\n\n⚠️ **Note:** Pour Gmail, utiliser un mot de passe d'application (pas le mot de passe du compte)\n\n## JWT Authentication\n\n### JWT_SECRET_KEY\n**Type:** String  \n**Description:** Clé pour signer les tokens JWT  \n**Défaut:** SECRET_KEY de Django\n\n```env\nJWT_SECRET_KEY=your-jwt-secret-key\n```\n\n### JWT_ACCESS_TOKEN_LIFETIME\n**Type:** Integer (secondes)  \n**Description:** Durée de vie du token d'accès  \n**Défaut:** 3600 (1 heure)\n\n```env\nJWT_ACCESS_TOKEN_LIFETIME=3600      # 1 heure\n```\n\n### JWT_REFRESH_TOKEN_LIFETIME\n**Type:** Integer (secondes)  \n**Description:** Durée de vie du token de refresh  \n**Défaut:** 604800 (7 jours)\n\n```env\nJWT_REFRESH_TOKEN_LIFETIME=604800   # 7 jours\n```\n\n## CORS\n\n### CORS_ALLOWED_ORIGINS\n**Type:** String (URLs séparées par virgule)  \n**Description:** Origines autorisées pour requêtes CORS\n\n```env\n# Développement\nCORS_ALLOWED_ORIGINS=http://localhost:3000,http://localhost:8000\n\n# Production\nCORS_ALLOWED_ORIGINS=https://yourdomain.com,https://api.yourdomain.com\n```\n\n## Cache & Session\n\n### REDIS_URL\n**Type:** String  \n**Description:** URL de connexion Redis (cache et sessions)\n\n```env\n# Développement (Redis local)\nREDIS_URL=redis://localhost:6379/0\n\n# Production (Redis distant)\nREDIS_URL=redis://user:password@redis.example.com:6379/0\n```\n\n## Pagination\n\n### PAGE_SIZE\n**Type:** Integer  \n**Description:** Résultats par page (API)  \n**Défaut:** 20\n\n```env\nPAGE_SIZE=20\n```\n\n## Logging\n\n### LOG_LEVEL\n**Type:** String  \n**Description:** Niveau de verbosité des logs  \n**Valeurs:** DEBUG, INFO, WARNING, ERROR, CRITICAL\n\n```env\n# Développement\nLOG_LEVEL=DEBUG\n\n# Production\nLOG_LEVEL=INFO\n```\n\n## Features Flags\n\n### ENABLE_COMMENTS\n**Type:** Boolean  \n**Description:** Activer le système de commentaires\n\n```env\nENABLE_COMMENTS=True\n```\n\n### ENABLE_VOTING\n**Type:** Boolean  \n**Description:** Activer le système de votes\n\n```env\nENABLE_VOTING=True\n```\n\n### ENABLE_PDF_EXPORT\n**Type:** Boolean  \n**Description:** Activer l'export PDF\n\n```env\nENABLE_PDF_EXPORT=True\n```\n\n### ENABLE_EMAIL_NOTIFICATIONS\n**Type:** Boolean  \n**Description:** Activer les notifications par email\n\n```env\nENABLE_EMAIL_NOTIFICATIONS=False\n```\n\n### ENABLE_WEBSOCKET_NOTIFICATIONS\n**Type:** Boolean  \n**Description:** Activer les notifications temps réel (WebSocket)\n\n```env\nENABLE_WEBSOCKET_NOTIFICATIONS=False\n```\n\n## Stockage des fichiers\n\n### MEDIA_URL\n**Type:** String  \n**Description:** URL publique pour les uploads\n\n```env\nMEDIA_URL=/media/\n```\n\n### MEDIA_ROOT\n**Type:** String  \n**Description:** Chemin local pour stocker les uploads\n\n```env\nMEDIA_ROOT=./media/\n```\n\n## AWS S3 (Optionnel)\n\n### USE_S3\n**Type:** Boolean  \n**Description:** Utiliser AWS S3 pour les fichiers\n\n```env\nUSE_S3=False\n```\n\n### AWS_ACCESS_KEY_ID\n**Type:** String  \n**Description:** Clé d'accès AWS\n\n```env\nAWS_ACCESS_KEY_ID=your_access_key\n```\n\n### AWS_SECRET_ACCESS_KEY\n**Type:** String  \n**Description:** Clé secrète AWS\n\n```env\nAWS_SECRET_ACCESS_KEY=your_secret_key\n```\n\n### AWS_STORAGE_BUCKET_NAME\n**Type:** String  \n**Description:** Nom du bucket S3\n\n```env\nAWS_STORAGE_BUCKET_NAME=civicfix-media\n```\n\n## Monitoring (Optionnel)\n\n### SENTRY_DSN\n**Type:** String  \n**Description:** URL Sentry pour error tracking\n\n```env\nSENTRY_DSN=https://your-key@sentry.io/project-id\n```\n\n## Elasticsearch (Optionnel)\n\n### ELASTICSEARCH_URL\n**Type:** String  \n**Description:** URL du serveur Elasticsearch\n\n```env\nELASTICSEARCH_URL=http://localhost:9200\n```\n\n## Admin\n\n### ADMIN_URL\n**Type:** String  \n**Description:** Chemin de l'interface admin\n\n```env\nADMIN_URL=admin/\n```\n\n## Fichier .env complet (template)\n\n```env\n# ============================================\n# DJANGO SETTINGS\n# DB_PORT=5432\n\n# ============================================\n# EMAIL\n# ============================================\nEMAIL_BACKEND=django.core.mail.backends.console.EmailBackend\n# EMAIL_HOST=smtp.gmail.com\n# EMAIL_PORT=587\n# EMAIL_USE_TLS=True\n# EMAIL_HOST_USER=your-email@gmail.com\n# EMAIL_HOST_PASSWORD=your-app-password\n\n# ============================================\n#
