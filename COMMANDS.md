# 📝 Commandes Django CivicFix\n\nGuide complet des commandes Django les plus utiles pour CivicFix.\n\n## Bases de données\n\n### Créer des migrations\n\n```bash\n# Créer les migrations pour tous les modèles modifiés\npython manage.py makemigrations\n\n# Créer les migrations pour une app spécifique\npython manage.py makemigrations apps.accounts\npython manage.py makemigrations apps.reports\npython manage.py makemigrations apps.dashboard\n\n# Voir les migrations qui seraient créées (dry-run)\npython manage.py makemigrations --dry-run\n\n# Générer un nom personnalisé\npython manage.py makemigrations --name add_custom_field\n```\n\n### Appliquer les migrations\n\n```bash\n# Appliquer toutes les migrations\npython manage.py migrate\n\n# Appliquer les migrations pour une app\npython manage.py migrate apps.accounts\n\n# Voir les migrations appliquées\npython manage.py showmigrations\n\n# Voir les migrations appliquées (une app)\npython manage.py showmigrations apps.reports\n\n# Annuler les dernières migrations\npython manage.py migrate apps.accounts 0001  # Revenir à la première migration\n\n# Voir le SQL généré\npython manage.py sqlmigrate apps.accounts 0001\n```\n\n### Interactions avec la BD\n\n```bash\n# Lancer la console de gestion BD\npython manage.py dbshell\n\n# Exporter les données\npython manage.py dumpdata > backup.json\npython manage.py dumpdata apps.reports > reports_backup.json\npython manage.py dumpdata --indent 2 > backup_formatted.json\n\n# Importer les données\npython manage.py loaddata backup.json\npython manage.py loaddata reports_backup.json\n\n# Flush la base (ATTENTION: supprime tout!)\npython manage.py flush\npython manage.py flush --noinput  # Sans confirmation\n```\n\n## Utilisateurs et Admin\n\n### Gestion des utilisateurs\n\n```bash\n# Créer un superutilisateur (admin)\npython manage.py createsuperuser\n\n# Créer un utilisateur sans interaction\npython manage.py createsuperuser --email=admin@example.com --username=admin --noinput\n# Puis changer le mot de passe:\npython manage.py shell\n# >>> from django.contrib.auth import get_user_model\n# >>> User = get_user_model()\n# >>> u = User.objects.get(username='admin')\n# >>> u.set_password('newpassword')\n# >>> u.save()\n\n# Changer le mot de passe d'un utilisateur\npython manage.py changepassword username\n```\n\n## Fichiers statiques\n\n```bash\n# Collecter les fichiers statiques\npython manage.py collectstatic\n\n# Sans demander de confirmation\npython manage.py collectstatic --noinput\n\n# Supprimer les anciens fichiers\npython manage.py collectstatic --clear --noinput\n\n# Voir ce qui serait collecté (dry-run)\npython manage.py collectstatic --dry-run --verbosity 3\n```\n\n## Tests\n\n```bash\n# Exécuter tous les tests\npython manage.py test\n\n# Tests d'une app\npython manage.py test apps.accounts\npython manage.py test apps.reports\npython manage.py test apps.dashboard\n\n# Tests d'une classe spécifique\npython manage.py test apps.accounts.tests.UserAuthenticationTests\n\n# Tests d'une méthode spécifique\npython manage.py test apps.accounts.tests.UserAuthenticationTests.test_user_registration\n\n# Tests avec verbosité\npython manage.py test --verbosity=2  #
