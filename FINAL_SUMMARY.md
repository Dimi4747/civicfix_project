## 🆘 Problèmes Courants

### **Port 8000 déjà utilisé**
```bash
python manage.py runserver 8001
```

### **Database locked**
```bash
rm db.sqlite3
python manage.py migrate
```

### **Module not found**
```bash
pip install -r requirements.txt
```

### **Static files missing**
```bash
python manage.py collectstatic --clear
```
